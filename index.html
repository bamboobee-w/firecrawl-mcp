🚀 How to Use

Send a POST request to /extract with the following:

{
  URL: https://firecrawl-mcp-lk7q.onrender.com/extract
  Method: POST
  Headers:
    Content-Type: application/json
    x-api-key: firecrawl-mcp
  Body:
    {
      "url": "https://en.wikipedia.org/wiki/OpenAI"
    }
}

📥 Example API Request and Response

Terminal Request (curl):
curl -X POST https://firecrawl-mcp-lk7q.onrender.com/extract \
  -H "Content-Type: application/json" \
  -H "x-api-key: firecrawl-mcp" \
  -d '{
    "url": "https://en.wikipedia.org/wiki/OpenAI"
}'
